자바스크립트에서의 데이터 입/출력

    * window : 자바스트립트 내장 객체로 브라우저 창이 열릴때마다 하나씩 만들어지는 객체

               브라우저 창 안에 존재하는 모든 요소들의 최상위 객체(생략가능)
    * document : 웹 문서마다 하나씩 만들어지는 객체(html 문서에 대한 정보들을 가지고 있음)

1. 데이터 출력하는 구문

1) [winodw.]alert("알림창에 출력할 문구");
2) [winodw.]console.log("콘솔에 출력할 문구");
3) document.write("화면상에 출력할 문구");
4) 선택한요소.innerHTML|innerText = "요소에 출력할 문구";

3) document.write("화면상에 출력할 문구");



4) 선택한요소.innerHTML|innerText = "요소에 출력할 문구";

자바스트립트에서 어떤 요소(element)안의 값(content)를 가지고 온다거나 변경하고자 할 때 innerHTML, innerText라는 속성에 접근


2. 데이터 입력받는 구문 (변수에 기록 가능)

1) 변수 = [window.]confirm("질문내용");
2) 변수 = [window.]prompt("질문내용");
3) 변수 = 선택한요소.속성(id, className, innerHTML, innerText ...);
4) 변수 = 선택한input요소.value;

1) 변수 = [window.]confirm("질문내용");

confirm호출시 "질문내용"과 "확인/취소"버튼이 존재하는 알림창 발생
확인 버튼을 클릭하면 true, 취소 버튼을 클릭하면 false를 반환


2) 변수 = [window.]prompt("질문내용");

prompt호출시 "질문내용"과 입력할 수 있는 "텍스트상자"와 "확인/취소" 버튼이 보여지는 알림창 발생
확인버튼 클릭시 텍스트상자에 입력한 값이 문자열로 반환, 취소버튼 클릭시 null값 반환
