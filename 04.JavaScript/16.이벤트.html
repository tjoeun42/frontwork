이벤트(event)

1. 이벤트 모델 종류

* 이벤트 속성에 핸들러 바인딩

특정 요소 객체를 가지고 와서 해당 요소의 이벤트 속성에 접근한 후 이벤트 핸들러를 연결하는 방식
이벤트를 제거할 수도 있음


* 인라인 이벤트 모델

요소내부에 직접적으로 이벤트 속성을 제시하여 실행할 내용을 정의하는 방식
주로 script태그 내에 정의되어 있는 함수 호출 방식 선호


* addEventListener에 등록

W3C에서 공식적으로 지정한 이벤트 모델

2. 현재 이벤트가 발생한 요소객체에 접근하기

태그별 기본적으로 가지고 있는 이벤트 제거

* 기본적으로 이벤트를 가지고 있는 요소

- a태그 : 클릭시 href의 속성값이 있는 url요청하는 기본 이벤트 보유
- submit버튼 : 클릭시 form에 들어있는 모든 값들을 서버로 전송

기본 이벤트 제거
=> 이벤트 핸들어의 return값을 false로 하면 기본이벤트 실행 안됨
