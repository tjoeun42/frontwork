window 객체

window 객체는 자바스크립트의 최상위 객체이며 크게 BOM과 DOM으로 나뉨
- BOM(Browser Object Model) : location, screen, navigator, history
- DOM(Document Object model) : document

- [window.]open('url', '창이름', '창의특성')


window 객체의 timer관련 메소드

- setTimeout(함수, 시간ms)

내가 제시한 일정 시간 후에 딱 한번만 실행


- [window.]setInterval(함수, 시간ms)

내가 지정한 시간 간격마다 "매번" 함수 실행

* DOM(Document Object Model)

HTML에 있는 각각의 요소들을 노드(Node)라고 함
> 요소노드(Element Node) : 태그 그 자체만을 의미
> 텍스트노드(Text Node) : 태그 내에 들어있는 내용
- 텍스트노드가 존재하는 요소(시작태그과 종료태그가 한쌍으로 이루어져 있음) : p, div,h~, li, a ... - 텍스트노드가 존재하지 않는 요소(시작태그로만 이루어져 있음) : br, hr, input, img ...

- 텍스트 노드가 존재하는 노드 생성 (시작태그 + 종료태그)

- 텍스트 노드가 없는 노드 생성 (시작태그)

- 노드(요소) 삭제와 관련된 메소드
