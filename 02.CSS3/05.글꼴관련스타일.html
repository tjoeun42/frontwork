font-size : 글꼴의 크기를 변경할 때 사용

        선택자 { font-size : 크기(px|em|rem|%)}

        * em : 부모 요소나 상속된 요소의 글꼴의 크기를 기준으로 크기 결정
        * rem : 루트 요소의 글꼴 크기를 기준으로 크기를 결정
        * % : 부모 요소의 글꼴 크기를 기준으로 크기를 결정, em과 유사
          > 기본 글꼴의 크기 : 16px
  • 고정크기 px테스트
  • 가변크기 em테스트
  • 가변크기 rem테스트
  • 가변크기 %테스트

font-weight : 글꼴의 두께 지정

        선택자 {
            font-weight:normal|bold|bolder|lighter|100|400|900;
        }
  • 원래 두께
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

font-variant : 영문 텍스트 문구를 작은 대문자로 변경할 때 사용

        선택자 { font-variant : small-caps; }

        선택자 {
            font-style: normal|italic|oblique;

font : 모든 속성을 한번에 사용

        선택자 {
            font: [font-style] [font-variant] [font-weight] [font-size|line-height] [font-family]
            font: bold 16px "맑은고딕", sans-serif;
        }
