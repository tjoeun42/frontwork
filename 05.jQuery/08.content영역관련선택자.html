Content 관련 메소드

* html(['문구']) 메소드 : innerHTML 속성과 관련된 메소드

선택된 요소의 content 영역의 값을 리턴해주거나 또는 변경해주는 메소드
> $('선택자').html() : 선택된 요소의 content영역 값을 반환해 줌(html태그도 포함)
> $('선택자').html('문구') : 선택된 요소의 content영역 값을 해당 문구로 변경해 줌(html태그가 있는 경우 태그로 인식함)

네이버로

* text(['문구']) 메소드 : innerText속성과 관련된 메소드

선택된 요소의 content영역의 값을 리턴해주거나 또는 변경해주는 메소드
> $('선택자').text() : 선택된 요소의 content영역 반환해줌(html태그는 포함안함, 텍스로만)
> $('선택자').text('선택자') : 선택된 요소의 content영역을 전달한 값으로 변경해줌(html태그는 포함안함, 텍스로만)
