strong : 글자를 굵게 표시하는 태그 + 스크린 리더

i : 글자를 기울여서 표시하는 태그

em(emphasis) : 특히 강조할 때 글자를 기울여서 표시하는 태그

mark : 형광펜 효과를 주는 태그

u : 글자 아래에 밑줄이 그어지는 태그

s : 글자에 취소선을 넣어주는 태그

small : 글자를 작게 표현해주는 태그

기본글자에 sub : 아래첨자를 나타내는 태그
sup : 위첨자를 나타내는 태그

abbr : 줄임말 표시
